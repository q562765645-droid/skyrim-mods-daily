# 优设文章收藏夹
# 格式：编号 | 标题 | 链接 | 标签 | 收藏日期 | 用户备注

# 收藏编号沿用推送编号，方便追溯
# 用户可以说："收藏 UY-260607-01-002" 或 "收藏 002" 或 "收藏 Codex那篇"

# 示例：
# UY-260607-02-002 | 案例实操！教你用GPT Image 2 出设计图 | https://www.uisdc.com/codex-9 | [GPT Image 2] [品牌设计] [案例] | 2026-06-07 | 想照着做一个潮牌网站

# 用法：
# 用户说 "收藏 编号" → 将该文章从已推送记录移到收藏夹
# 用户说 "收藏 关键词" → 匹配最近推送中标题/标签含该关键词的文章
# 用户说 "取消收藏 编号" → 从收藏夹移除
# 用户说 "我的收藏" → 列出所有收藏
# 用户说 "收藏夹 标签" → 按标签筛选收藏

# 标签体系（与推送一致）：
# [Codex] [Figma] [GPT Image 2] [Claude] [Claude Code] [Skill] [Agent] [MetaSkill]
# [UI走查] [前端还原] [品牌设计] [海报设计] [AI绘画] [Vibe Coding] [编程]
# [交互设计] [产品分析] [面试] [职业] [创业] [团队管理] [效率]
# [自动化] [工具教程] [案例] [观点] [趋势] [新功能] [设计系统]
# [字体] [配色] [排版] [3D] [动效] [插画] [图标] [作品集]
# [Midjourney] [Stable Diffusion] [即梦] [可灵] [Sora] [Suno]
